# CdsCTF Kubernetes Helm Charts

## Usage

[Helm](https://helm.sh) must be installed to use the charts.
Please refer to Helm's [documentation](https://helm.sh/docs/) to get started.

Once Helm has been set up correctly, add the repo as follows:

```
helm repo add cdsctf https://cdsctf.github.io/helm-charts
```

You can then run `helm search repo cdsctf` to see the charts.

Please refer to CdsCTF's [documentation](https://cdsctf.e23.dev) to learn more about CdsCTF.
